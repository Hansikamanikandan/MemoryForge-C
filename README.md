# 🚀 Project Overview

**MemoryForge-C** explores one of the most powerful features of the C programming language—**Dynamic Memory Allocation**.

This repository demonstrates how memory can be allocated, accessed, resized, and released during runtime using pointers. The programs focus on efficient memory utilization while solving practical problems involving arrays, searching, and string processing.

# 📚 Dynamic Memory Functions Used

| Function | Purpose |
|----------|---------|
| `malloc()` | Allocates memory dynamically. |
| `calloc()` | Allocates and initializes memory with zero. |
| `realloc()` | Resizes previously allocated memory. |
| `free()` | Releases dynamically allocated memory. |
| `sizeof()` | Returns the size of a data type or variable. |
